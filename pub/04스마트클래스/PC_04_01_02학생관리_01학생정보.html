학생 일괄 등록 안내
  1. 엑셀양식을 다운로드 받습니다.
  2. 다운로드 받은 양식을 엑셀필드 설명에 맞게 입력합니다.
    - 필수 등록항목 : 아이디, 비밀번호, 닉네임
    * 각 항목은 입력 시 빈 공간 없이 입력하셔야 합니다.
    * 추후 원활한 학생 검색을 위해 영어 이름으로 학생 ID 생성 권장 합니다. (반에 학생 추가 시, ID 검색으로만 추가 가능합니다.)
  3. 엑셀 업로드창에 입력한 엑셀파일을 업로드 합니다.
    - 엑셀 업로드 시 우선 학생 한 명만 샘플로 등록하여 정상적으로 입력되는지 확인 후 전체 학생목록을 업로드 하세요.
    - 아이디가 기존 회원과 중복되는 경우 해당 학생 정보를 제외하고 시스템에 업로드 됩니다.
  4. 학생 정보가 정상적으로 등록 되었는지 확인합니다.
